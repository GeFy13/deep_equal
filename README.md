# Deep Equal Function Project

This project implements a custom `deepEqual` function for deep comparison of objects in JavaScript, along with comprehensive Jest tests.

## Features

- Compares primitive values
- Compares nested objects
- Compares arrays (including nested arrays)
- Compares Date objects
- Compares RegExp objects
- Handles different key orders in objects
- Compares Map and Set objects
- Handles Symbol properties

## Installation

```bash
npm install
```

## Run test

```bash
npm test
```
